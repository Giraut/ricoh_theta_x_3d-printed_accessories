## Assembly

### Base

This base stand can be screwed onto any surface or heavy object to turn it into a secure base for the Ricoh Theta X camera. The surface only needs 2 threaded M3 holes 35mm apart - or straight 3mm-diameter holes if you use nuts - for the countersunk head M3 screws.

![Base assembly example #1](images/ricoh_theta_x_3d-printed_base_assembly_example1.jpg)
![Base assembly example #2](images/ricoh_theta_x_3d-printed_base_assembly_example2.jpg)
![Base assembly example #3](images/ricoh_theta_x_3d-printed_base_assembly_example3.jpg)

### Base with Benro selfie stick mount

- Unscrew the stock swiveling head fully and detach it from the end of the selfie stick

![Base with Benro selfie stick mount - assembly #1](images/ricoh_theta_x_3d-printed_base_with_benro_selfie_stick_mount_assembly1.jpg)

- Slip the base onto the end of the selfie stick

![Base with Benro selfie stick mount - assembly #2](images/ricoh_theta_x_3d-printed_base_with_benro_selfie_stick_mount_assembly2.jpg)

- Lock the base in place with a 5mm screw

![Base with Benro selfie stick mount - assembly #3](images/ricoh_theta_x_3d-printed_base_with_benro_selfie_stick_mount_assembly3.jpg)
